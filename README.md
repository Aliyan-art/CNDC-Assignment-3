### Setup & Execution Instructions
1. Download the `app.py` file.
2. Run the server on your Windows laptop terminal using: `python app.py`
3. Connect your iPhone 13 Pro Max to the same Wi-Fi network.
4. Open Safari on your iPhone and type the host IP address followed by the port: `http://<YOUR_LAPTOP_IP>:8080`
